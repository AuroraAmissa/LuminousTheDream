This directory needs to contain the .ttf files from the following fonts:

* https://fonts.google.com/download?family=Cormorant%20Infant
* https://fonts.google.com/download?family=Dancing%20Script
* https://fonts.google.com/download?family=Fira%20Sans
* https://fonts.google.com/download?family=Quicksand

The following fonts should be present in this directory after downloading:

* CormorantInfant-Bold.ttf
* DancingScript-Bold.ttf
* DejaVuSansMono.ttf
* FiraSans-BoldItalic.ttf
* FiraSans-Bold.ttf
* FiraSans-Italic.ttf
* FiraSans-Regular.ttf
* Quicksand-Medium.ttf

Alternatively, get the fonts from the full source code here: https://github.com/Lymia/LuminousTheDream
